## サボテン

2026/4/19 記録
&nbsp;

![サボテン](/images/サボテン.jpg)

私の家には、一つの鉢に寄せ植えされたサボテンがあります。ある日、散歩がてらに寄ったコーナンで買ってきました(笑)。
ウチワサボテンの方の二つは、大きい方から「うさみみ一号」「うさみみ二号」、丸いサボテンは「サボ」と名付けています。

<br/>
![サボテンの寄せ植え](/images/サボの寄せ植え.jpg)
<span className="text-center-force" style={{display: "block", fontSize: "var(--text-sm)", color: "var(--color-black-soft)"}}>
サボテンの寄せ植え
</span>
<br/>

「うさみみ一号」の方が、とんでもなく「うさみみっぽい」大きな耳みたいなものがついています。これが結構かわいい。
やっぱり多肉植物と言うか、サボテン良いな～と言うことで、鉢に植わったサボテンを作ろうと思い、制作に取り掛かりました。
<br/>
最初は、鶴の基本形に領域を少し足したようなものに、立体的な鉢を付ける方向性で考えていました。
それっぽいものは一応完成したのですが、鉢の中のスカスカ感が拭えないことと、一方向からの作品になってしまいそうだったので没になりました。

<br/>
そこで、立体の鉢を制作することはやめて、両面どちらからみてもサボテンとわかるもの、かつどちらかと言うと平面っぽいものを目指してみました。ガラッと構造を変えてみて、一対の対角のカド周辺が鉢の部分、もう一対のカドがサボテンの育ってきたときの子株を表現する設計になりました。

![サボテンの展開図](/images/サボテン_展開図.png)
<span className="text-center-force" style={{display: "block", fontSize: "var(--text-sm)", color: "var(--color-black-soft)"}}>

<br/>

ある程度、自分の好きなところに子株を配置できますし、形も好みによって変えることができます。
鉢の横の部分を少しふっくらさせることにより自立可能で、絵に出てくるような「the サボテン」のようなデザインがよく似合います！
